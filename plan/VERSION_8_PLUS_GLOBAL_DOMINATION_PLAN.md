## Version 8 - Verified Competency Engine (Trust Layer)

### Goal
Make competency results credible, auditable, and role-relevant.

### Build
- Evidence scoring model:
  - lesson evidence
  - lab evidence
  - project evidence
  - peer/manager validation
- Confidence score per skill (`self_claimed`, `learned`, `verified`)
- Role-based target competency profiles (DevOps Intern/Fresher/Junior/Mid/Senior)
- Skill decay + recertification reminders
- Audit trail for all competency updates

### Why this wins
Most products track completion. You track **verified capability**.
